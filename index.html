Mod Installation & Requirements

Thank you for downloading! To make sure everything works properly, please follow the steps below carefully.

Required Files

These mods require the following libraries to function:

  • 🌐 Lot 51's Core Library
  • 📁 midnitetech_modlibrary

Make sure both are installed in your Mods folder and that you have the latest version.

🎮

Installation Tips

✅ Correct:
Mods/MyModFolder/MyMod.ts4script

❌ Incorrect:
Mods/MyModFolder/AnotherFolder/MyMod.ts4script

All .ts4script files must be no more than one folder deep inside your Mods folder.

Do not try and open these files — just drop them in your Mods folder.
