Required Files

99% of my mods require the following libraries to function:

  • 🌐 Lot 51's Core Library
  • 📁 midnitetech_modlibrary

Make sure both are installed in your Mods folder and that you have the latest version of each. All .ts4script files must be no more than one folder deep inside your Mods folder, as shown below.

✅ Correct:
Mods/MyModFolder/MyMod.ts4script

❌ Incorrect:
Mods/MyModFolder/AnotherFolder/MyMod.ts4script
🎮

Required DLC

Some of my mods require specific DLC packs. If you don’t have the required packs installed, you may see some Last Exception errors—they shouldn’t break your game, but please use with caution.

All of my mods are compatible with each other and with most other mods — the only exception is if two mods override the same function (e.g., Styled Look Hider).

Anything not listed below is fully Base Game compatible.
